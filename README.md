# Efficiency Demonstration Of Embedding Models And Libraries For Bengali Word Vector Representation

## Technology used in this project
- Python
- Regular expression
- Gensim
- FastText
- Pandas

## Project Goal
- Preprocess Text data
- Word Embedding
- Semantic, Syntactic and Word analogy

## Project details
- Collacting Data using Scrapy
- Read data using pandas
- Data preprocess using regular expression
- Training Different models(FastText, Gensim, Glove)
- Evaluation through Semantic, Syntactic and Word analogy
